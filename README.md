# jellyfin2link


This website can display your jellyfin media library and provide api endpoints to run media from your jellyfin instance.

Auto grabs the best compatabile codecs that are supported on both streaming and receiving clients. (UI for this needs to be done. It is hard coded for [Resonite](https://resonite.com/))

Currently only works for video media, others are planned in the future.

# Setup
1. Create an api key in Jellyfin.
2. Edit jellyfin2link/appsettings.json
3. Update the Jellyfin fields and follow the format.
